# homelab-controller

An open-source, low-cost home-lab controller focused on custom features and tools.

## Why this project?

This project provides a cheaper, self-hosted controller experience for homelabs:

- Open by default (no vendor lock-in)
- Built for commodity hardware
- Extensible with custom operational tools

## Current capabilities

- Register unique custom features for your lab workflow
- Register custom tools with command templates
- List configured features and tools from the CLI

## Run locally

```bash
cargo run
```
